# SHOW MARKS

show marks on signcolumn for vim or nvim

fast, mini, simple

## INSTALL

`Plug 'yaocccc/vim-showmarks'`

## CONFIG

```viml
  hi Marks ctermfg=80
  let g:mark_ns_id = [number]  " default 9898
  let g:mark_priority = [number] " default 999
```
